# vue-Namer
##变量命名器
***

项目需要引入你自己的百度API的appid和key
在mianContent.vue文件中添加即可
```
data() {
      return {
        fromValue: '',
        toValue: '',
        loading: false,
        appid: '',   //添加你自己的百度翻译API的appid
        key: '',    //添加你自己的百度翻译API的key
        Type: 'Camel'
      }
    }
```
